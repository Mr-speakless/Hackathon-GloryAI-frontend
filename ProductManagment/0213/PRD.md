# PRD – AI 护肤新手指南（Hackathon Version）

## 一、产品概述

**产品名称（暂定）**：GLoryAI 可以改一改家人们

**产品定位**：一款面向护肤小白的 AI 护肤入门指南工具

**核心价值**：3分钟完成皮肤检测 + 自动生成专属护肤方案

基于 **Perfect Corp YouCam AI Skin Analysis API** 实现面部皮肤问题检测（支持14项皮肤问题识别，如 acne、wrinkle、dark circle 等）。

---

## 二、目标用户

- 18–30岁护肤小白
- 不知道自己皮肤类型
- 不懂护肤产品怎么选
- 害怕踩雷

---

## 三、产品目标（Hackathon阶段）

1. 集成至少 1 个 Perfect Corp API（AI Skin Analysis）
2. 生成自动化皮肤报告
3. 根据问题推荐入门护肤方案
4. 展示清晰的用户价值（新手友好 + 决策辅助）

---

# 四、产品功能 1.0

## 4.1 用户流程

上传照片

→ 调用 Perfect Corp Skin Analysis API

→ 获取 JSON 分析结果

→ 生成皮肤问题报告

→ 推荐护肤产品组合

---

## 4.2 功能模块清单

### 1️⃣ 图片上传接口

- 支持 JPG / PNG
- 前端预览
- 调用后端 API

---

### 2️⃣ Perfect Corp API 接入

调用：

- AI Skin Analysis API（支持14项皮肤问题检测）

返回数据示例：

- acne score
- wrinkle score
- dark circle
- pore
- redness
- oiliness等

---

### 3️⃣ 动态皮肤报告生成

输出内容：

- 皮肤类型判断（干性/油性/混合）
- Top 3 皮肤问题
- 风险等级提示
- 新手解释（通俗语言）

示例：

> 你的皮肤偏油，存在轻度痘痘和毛孔问题，建议从基础控油 + 温和修护开始。
> 

---

### 4️⃣ 护肤产品数据库（初版）

规则：

每个皮肤问题 → 推荐 2 款产品

数据库结构：

| 问题 | 产品1 | 产品2 |
| --- | --- | --- |
| Acne | A洗面奶 | B精华 |
| Wrinkle | A抗老精华 | B面霜 |

---

### 5️⃣ 推荐等级系统

根据用户需求输出三种版本：

| 等级 | 推荐数量 | 适用人群 |
| --- | --- | --- |
| 新手版 | 3 个产品 | 极简护肤 |
| 进阶版 | 4 个产品 | 有基础 |
| 高手版 | 6 个产品组合 | 完整流程 |

逻辑：

根据 API 分数排序 → 匹配问题 → 推荐组合
